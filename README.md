# os-guide

Operating Systems study guide — beginner to interview-level, hosted as a single self-contained HTML page.

**Live site:** https://lakshmipriyaanbu.github.io/os-guide/

## Updating the page

```bash
cp <new-file> ~/Desktop/os-guide/index.html
cd ~/Desktop/os-guide && git add index.html && git commit -m "update" && git push
```

GitHub Pages redeploys automatically within a minute of the push.
